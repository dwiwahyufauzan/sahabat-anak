# Sahabat Anak Indonesia

Website yayasan nirlaba yang berfokus pada kesejahteraan dan pendidikan anak-anak marjinal di Indonesia.

## Tech Stack

- SvelteKit
- Tailwind CSS
- Svelte 5

## Development

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
```

## Deploy

Project ini otomatis di-deploy ke GitHub Pages saat push ke branch `main`.
